O posicionamento

O fluxo normal das caixas é um block-level a abaixo do outro. Isso é devido ao esquema padrão de posicionamento: "position: static;".

Ao todo são quatro esquemas de posicionamento: static; relative, absoluto e o fixed. Com a exceção de static é possível mover os outros tres nas direções para cima, para baixo, para a direita e para a esquerda. Vejamos cada um deles.
