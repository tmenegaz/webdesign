Strings

Uma string em JavaScript são sequências de caracteres de 16 bits sem sinal.

As strings são como arrays de caracteres, isto é, cada caracter é idenxado e são imutáveis.

Os valores primitivos de string são comparados como iguais se, e somente se, tiverem o mesmo comprimento e se o coracter em cada índice for o mesmo

Resposta da comparação:

As strings pode ser tratadas como arays somente para leitura e é possível acessar caracteres individuais de uma string usando colchetes ou o método charAt();

Concatenar tipos diferentes

É possível por meiodo operador de adição (+) adicionar uma string a um número

Cast

Use Number() para transformar uma string em número e toString() para o contrário

String como Objeto

Tudo é um Objeto em JavaScript! Os valores de uma string podem ser também primitivos e nunca são mutáveis (os Objetos são mutáveis). Acontece que as Strings são convertidas em Objetos wrapper para herdar as propriedades de String

Mas também pode retornar para sua parte primitiva com valueOf()
