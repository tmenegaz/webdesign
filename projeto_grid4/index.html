  • Entrega: 2/12/2019
  • Assunto: layout CSS - grid e flex
  • Valor: 10 pontos

A fim de concluir este site o estudante deverá elaborar 3 páginas com layout de duas colunas e uma lista de navegação lateral com os itens dispostos verticalmente. Este site támbem serve como referência sobre como usar o grid e o flex, além dos linques no footer. Essas páginas serão acessadas pelo linques de navegação principal para página 1, 2, 3 e o home. Uma vez que home seja acessado, a página home devera ser exibida, assim como as demais quando acessadas com seu respectivos conteúdos (Ex: essa é a página x). O top, nav e footer são iguais ao da home para todas as páginas. Usar css externo.
